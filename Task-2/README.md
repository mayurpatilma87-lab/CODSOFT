# 🎬 Movie Rating Prediction with Python

## 📌 Project Overview
Movie Rating Prediction is a Machine Learning project that predicts the rating of a movie based on features such as genre, director, and actors. The project uses regression techniques to analyze historical movie data and estimate movie ratings accurately.

This project demonstrates data preprocessing, feature engineering, and machine learning model building using Python.

---

## 🎯 Objectives
- Analyze movie dataset to understand factors affecting ratings
- Perform data cleaning and preprocessing
- Convert categorical data into numerical format
- Build a regression model to predict movie ratings
- Evaluate model performance

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Google Colab / Jupyter Notebook

---

## 📂 Dataset
The dataset contains movie details such as:
- Movie Name
- Genre
- Director
- Actors
- Year
- Rating (Target Variable)

Dataset used: **IMDb Movies India Dataset**

---

## ⚙️ Project Workflow

1. Import required libraries
2. Load dataset
3. Data cleaning and preprocessing
4. Handle missing values
5. Convert categorical data to numeric (Label Encoding)
6. Split dataset into training and testing data
7. Train model using Linear Regression
8. Predict movie ratings
9. Evaluate model performance

---

## 🤖 Machine Learning Model
- Linear Regression Algorithm
- Used to predict movie ratings based on input features

---

## 📊 Evaluation Metrics
- R2 Score
- Mean Squared Error (MSE)
